Peds Viewer - программа для просмотра педов из GTA Vice City, включая кутсценных.
Cделана по исходникам Moo Mapper 0.90, не требует установки.
Программа для тех, кто занимается изменением городского трафика.
Для удобства все педы разделены на группы, можно легко узнать как выглядит каждый пед и его название.
Бонусом добавлен просмотр оружия.
Для просмотра моделей отмечаем справа загружаемые файлы, слева выбираем список моделей и ниже выбираем конкретную модель.
